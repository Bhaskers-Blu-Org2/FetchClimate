Solutions overview.

Solutions that do not require additional visual studio plugins:

DataHandlers - Contains particular final data handlers that can be applied to particular datasets.
FetchEngine - Contains FetchEngine project and its dependencies
FetchConfig - Contains FetchConfig.exe project and its dependencies
AzureRoles - Contains a code for AzureWebRole(FetchClimate Frontend) and AzureWorkerRole (FetchWorker)

Solutions that require additional visual studio plugins:

Databases - Contains projects for configuration database and Jobs databases of fetchclimate servicer. Requires DataBase tools for visual studio
AzureDeployment - Contains deployment project to publish fetch climate service the Microsoft Windows Azure. Requires Azure SDK to be installed
